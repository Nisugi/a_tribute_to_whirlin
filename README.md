# Whirlin Trainer Web Application

A static web application converted from the Whirlin_Trainer_2025.1.VBA.xlsm Excel spreadsheet.

## Overview

This project extracts data and functionality from the original Excel spreadsheet with VBA macros and converts it into a standalone web application using HTML, CSS, and JavaScript.

## Features

- Character training calculator
- Ascension tracking
- Enhancive builds and inventory management
- Defense/Attack/Aiming strength calculators
- Spell effects reference
- Player skills manager

## Technology Stack

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages for hosting

## Structure

```
/
├── index.html          # Main application page
├── css/                # Stylesheets
├── js/                 # JavaScript modules
├── data/               # Extracted spreadsheet data (JSON/CSV)
└── assets/             # Images and other assets
```

## Development

This is a static site - simply open `index.html` in a web browser to run locally.

## Deployment

Hosted via GitHub Pages at: https://nisugi.github.io/a_tribute_to_whirlin/

The site automatically deploys when changes are pushed to the master branch.
